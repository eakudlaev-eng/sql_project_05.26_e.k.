# Справочник данных

В проекте используются синтетические данные по CRM-коммуникациям банка.

## `clients`

Клиентская база.

| Поле | Описание |
|---|---|
| `client_id` | идентификатор клиента |
| `gender` | пол |
| `age` | возраст |
| `city` | город |
| `segment` | клиентский сегмент |
| `registration_date` | дата регистрации |
| `is_active` | признак активности клиента |
| `last_activity_at` | дата последней активности |

## `campaigns`

Справочник CRM-кампаний.

| Поле | Описание |
|---|---|
| `campaign_id` | идентификатор кампании |
| `campaign_name` | название кампании |
| `channel` | канал коммуникации |
| `product` | продукт |
| `start_date` | дата старта |
| `target_segments` | целевые сегменты |

## `campaign_audience`

Аудитория кампаний.

| Поле | Описание |
|---|---|
| `audience_id` | идентификатор записи аудитории |
| `campaign_id` | идентификатор кампании |
| `client_id` | идентификатор клиента |
| `group_type` | группа клиента: test или control |
| `assigned_at` | дата попадания клиента в аудиторию |

## `messages`

Отправленные сообщения.

| Поле | Описание |
|---|---|
| `message_id` | идентификатор сообщения |
| `campaign_id` | идентификатор кампании |
| `client_id` | идентификатор клиента |
| `channel` | канал сообщения |
| `sent_at` | время отправки |
| `status` | статус сообщения |

## `message_events`

События по сообщениям.

| Поле | Описание |
|---|---|
| `event_id` | идентификатор события |
| `message_id` | идентификатор сообщения |
| `event_type` | тип события: sent, delivered, opened, clicked |
| `event_at` | время события |

## `loan_applications`

Заявки после коммуникаций.

| Поле | Описание |
|---|---|
| `application_id` | идентификатор заявки |
| `client_id` | идентификатор клиента |
| `campaign_id` | идентификатор кампании |
| `product` | продукт |
| `application_at` | время заявки |
| `status` | статус заявки |
| `requested_amount` | сумма заявки |
